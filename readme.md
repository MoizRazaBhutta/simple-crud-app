# CRUD Express JS Application

This is a simple CRUD application built using Express JS and MongoDB.

## Installation

To install this application, please follow the steps below:

1. Clone the repository by running the command `git clone https://github.com/your-username/crud-express-js.git`
2. Navigate to the project directory by running the command `cd crud-express-js`
3. Install the dependencies by running the command `npm install`
4. Start the application by running the command `npm run dev`

## Usage

The application supports the following endpoints:

1. `GET /api/products`: Fetches all products from the database.
2. `GET /api/products/:id`: Fetches a product with the given ID from the database.
3. `POST /api/products`: Creates a new product in the database.
4. `PUT /api/products/:id`: Updates a product with the given ID in the database.
5. `DELETE /api/products/:id`: Deletes a product with the given ID from the database.

## Technologies Used

The following technologies were used to build this application:

1. Express JS: A Node.js web framework used to build web applications.
2. MongoDB: A NoSQL database used to store data.
3. Mongoose: A Node.js library used to interact with MongoDB.
4. Nodemon: A development tool used to automatically restart the application when changes are made to the code.
